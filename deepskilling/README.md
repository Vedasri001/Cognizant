# Cognizant Digital Nurture 5.0 – Deep Skilling

## DotNet Full Stack Engineer (React Track)

Welcome to the **Deep Skilling** section of my Cognizant Digital Nurture 5.0 repository. This directory contains all the hands-on exercises, projects, and assignments completed during the DotNet Full Stack Engineer (React Track) training program.

The modules are organized sequentially to reflect the learning path followed throughout the program, covering software engineering principles, backend development, frontend development, testing, microservices, and version control.

* Observer Pattern

* Command Pattern
